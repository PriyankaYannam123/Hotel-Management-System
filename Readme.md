# Hotel Management System (C++)

## Description:
This **C++** application manages the basic functionalities of a hotel, including **room bookings**, **room availability**, and **payment management**. The application uses **object-oriented programming** principles such as **inheritance**, **polymorphism**, and **encapsulation** to maintain scalability and ease of maintenance.

## Technologies Used:
- C++
- Object-Oriented Programming (OOP)
- Basic file handling for data storage

## Features:
- Manage hotel room availability and bookings
- Supports multiple rooms and allows for room status checks
- Basic payment processing for room booking

## How to Run:
1. Clone the repository.
2. Compile the `HotelManagementSystem.cpp` file.
3. Run the executable to interact with the hotel booking system.
